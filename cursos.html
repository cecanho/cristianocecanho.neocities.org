O que são cursos de extensão?

Certificados

Certificados digitais comprovando a habilidade adquirida com a quantidade de horas cursadas.

Como obter o certificado digital?

Primeiro é necessário realizar uma atividade que pode ser um questionário, ou atividade realizada no curso.

Onde consigo e envio esta atividade?

As atividades devem ser requeridas no e-mail: cecanho@gmail.com e enviados prints (recortes de tela pedidos no e-mail de retorno) para o mesmo e-mail.

Quanto tempo demora entre o curso e o retorno do certificado?

Você estuda no seu tempo. A correção da atividade enviada pode demorar em torno de uma semana, assim que corrigida e o pagamento confirmado o envio é imediato.

Como pagar e quanto custa?

A emissão do certificado depende do recolhimento via PIX (chave: cecanho@gmail.com) no valor de R$ 30,00 por curso/pacote.

Planta baixa em 2D com AutoCAD

Aprenda a construir a planta baixa de um quarto em 2D com o AutoCAD. Inicialmente este curso foi criado em parceria com a UNICEP, mas hoje não tem mais vínculo com a instituição. Com os conceitos apresentados neste curso você será capaz de criar plantas mais elaboradas. Carga horária 8 horas.

O que aprenderei neste curso:

  • Apresentação da estrutura do curso e download do software.
  • Desenhando paredes e porta.
  • Desenhando uma janela.
  • Uso de cotas.
  • Uso de camadas (Layers).

Agenda de Contatos com Java e MySQL

Já pensou em começar a programar em uma linguagem de programação com tipagem forte? Neste curso você aprende a programar em Java utilizando ferramentas livres junto com persistência de dados em MySQL com um total de 30 horas. Inicialmente este curso foi criado em parceria com a UNICEP, mas hoje não tem mais vínculo com a instituição.

O que aprenderei neste curso:

  • Apresentação da estrutura do curso.
  • Criação de um modelo para a classe Contato.
  • Criação de um controlador para a classe Contato.
  • Criação de uma fábrica de conexão com o MySQL.
  • Importância da organização do código em pacotes.
  • Criação de serviços.
  • Apresentação de dados via Terminal.
  • Como instalar e utilizar o WindowBuilder em seus projeto Java.

Pacote Microsoft Office

Neste pacote de cursos como Microsoft Word, Microsoft Excel e Microsoft PowerPoint com um total de 40 horas, o aluno aprende os seguintes recursos:

  • Microsoft Word: Formatar fonte, formatar parágrafo, formatar/inserir tabela, formatar/inserir imagem, mala direta, configurar página e exportar arquivo.
  • Microsoft Excel do básico ao avançado: Criar um calendário comentado, tipos de dados em Excel, criar uma planilha financeira, criar uma planilha para gestão de cartão de crédito, uso de funções matemáticas, SOMA, SOMASE, CONTSE, SE, PROCV, Macro e gráficos.
  • Microsoft PowerPoint para apresentações acadêmicas: Formatar fonte, formatar parágrafo, formatar/inserir tabela, formatar/inserir imagem, configurar página e exportar arquivo.
